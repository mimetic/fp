To insert an uploaded file in your text, write the file name and link title with [[ and ]] around them, and a ':' between, like this:&lt;div align='center'&gt;&lt;b&gt;[[my_file.jpg:My Picture]]&lt;/b&gt;&lt;/div&gt;Be sure to use the uploaded file name, which may differ from your original file name.&lt;br&gt;&lt;br&gt;&lt;B&gt;Examples:&lt;/b&gt;&lt;br&gt;&lt;i&gt;Picture:&lt;/i&gt; Here is my picture: [[my_picture.jpg]]&lt;br&gt;&lt;i&gt;PDF file:&lt;/i&gt; Here is my resume: [[my_resume.pdf:Click to see my resume]]&lt;br&gt;&lt;i&gt;Audio:&lt;/i&gt; Listen to [[birds_singing.mp3:the sound of birds singing]]&lt;br&gt;&lt;i&gt;Video:&lt;/i&gt; Watch the event: [[video_of_event.mov:A Happy Day]]&lt;br&gt;
