Web fonts require some code to be used in the HEAD of your web page. Different systems use different codes.  <i>Copy this code from your web fonts source, e.g. Google Fonts.</i><br><br>For example, Google Fonts uses something that looks like this:<br><small>&lt; link href= &quot;http://fonts.googleapis.com/css?family=Simonetta:400,900,400italic&quot; rel=&quot;stylesheet&quot; type=&quot;text/css&quot;&gt;</small><br><br>TypeKit uses a different code, which might look like this:<br><small> &lt; script type=&quot;text/javascript&quot; src=&quot;//use.typekit.net/ury6ybo.js&quot; &gt; &lt; /script &gt; &lt;script type=&quot;text/javascript&quot;&gt;try{Typekit.load();}catch(e){}&lt;/script&gt;</small>
